## 2.SNS回覧板とは
- 自治会の回覧板を、スマートフォンで運用する方法です。
- ①「LINE アプリ」と、②「Google ドライブ」の、組み合わせです。
- ファイル共有やアンケート機能をGoogleドライブで補います。
- 「グループチャット」よりも、プライバシーに配慮した方法です。
- 無料もしくは月額数千円での運用を想定しています。
- やり方を覚えてしまえば、自治会に限らず、応用が可能です。
